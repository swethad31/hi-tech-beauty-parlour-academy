# HI-TECH Bridal Studio and Academy 💄✨

A luxury, high-end responsive web application for **Hi-Tech Bridal Studio and Academy**, located in Puducherry. Built with a focus on premium aesthetics, portfolio protection, and interactive user experience.



## 🌟 Features

- **Luxury Branding**: Dark-themed UI with Gold (#D4AF37) and Crimson Red (#8B0000) accents.
- **Interactive Portfolio**: Masonry-style gallery with tabbed filtering (Portraits, Makeovers, Nail Art, etc.).
- **Smart Lightbox**: Full-screen image preview with high-quality resolution.
- **Content Protection**: 
  - **Screen Guard**: Transparent overlays to prevent right-clicking and "Save Image As."
  - **Interaction Shield**: Context menu disabling to protect proprietary bridal and nail art designs.
- **Responsive Navigation**: Custom red navbar with quick-access icons for Instagram, Location, and Reviews.
- **Dynamic Hero Section**: 3-grid hero layout with interactive "zoom-out" effects on pointer hover.
- **Service Catalog**: 8 specialized beauty and hair care categories with 3D-tilt hover effects.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
